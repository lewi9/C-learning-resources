Napisz dwa programy:
Pierwszy program przyjmuje parametr x,e i n:
x - liczba, którą zostanie zainicjalizowany generator liczb pseudoloswych, > 0 X
e - liczba dopuszczalnych błędów z rzędu podczas tworzenia pliku E
n - liczba plików, > 0 N
Program ma napoczątku wylosować swój identyfikator rand()%X
Następnie ma tworzyć pliki z flagą O_EXCL, jeśli się to nie powiedzie E razy z rzędu program kończy działanie
Tworzone pliki mają mieć nazwę file_J, gdzie J należy do zbioru [1,N]
Program ma zapisać do pliku swój identyfikator X jako tekst
Po utworzeniu pliku program ma odczekać 0.01 sekundy
Po zakończeninu działania program ma zwrócić swój indetyfikator

Drugi program przyjmuje dwa parametry p, n, e:
p - liczba utworzonych potomków, > 0 P
n - liczba plików, > 0 N
e - liczba dopuszczalnych błędów z rzędu podcza tworzenia pliku E
Program ma zainicjalizować generator liczb pseudolosywych srand(time(NULL))
Program ma utworzyć P potomków
Każdy z potomków ma stać się pierwszym programem, gdzie x = |rand()| + 1, e = E, n = N
Kiedy wszyscy potomkowie zakończą działanie, program ma zliczyć ile plików wygenerował każdy z potomków, a następnie usunąć pliki (unlink())
Program ma wyświetlić informacje w postaci: <pid_procesu>\t<identyfikator_procesu>\t<liczba_plików>
